# lib

## Purpose
Core utility modules: database client, authentication, session management, and helper functions.

## Key Files
| File | Description |
|------|-------------|
| `db.ts` | Prisma client singleton (prevents multiple instances in dev) |
| `auth.ts` | `getCurrentUser()` - reads session, returns user or null |
| `session.ts` | JWT session management (create, verify, delete cookies) |
| `utils.ts` | `cn()` - Tailwind class merging utility (clsx + tailwind-merge) |
| `time-ago.ts` | Relative time formatting ("2 saat önce") in Turkish |

## Subdirectories
| Directory | Purpose |
|-----------|---------|
| `generated/` | Auto-generated Prisma client (DO NOT EDIT) |

## For AI Agents

### Working In This Directory
- `db.ts` exports both named `prisma` and default export
- `auth.ts` depends on `session.ts` for JWT handling
- `utils.ts` is used by almost every component via `cn()`
- NEVER manually edit files in `generated/` — they are auto-generated by Prisma

### Common Patterns
- Import DB: `import prisma from "@/lib/db"`
- Import auth: `import { getCurrentUser } from "@/lib/auth"`
- Import cn: `import { cn } from "@/lib/utils"`

## Dependencies

### External
- `@prisma/client` - Generated database client
- `jose` - JWT token handling (in session.ts)
- `clsx` + `tailwind-merge` - Class name utilities
